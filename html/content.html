Website content

  1. vision what is website content?
    Tells what website content command does.

  2. Vision initiate web content
    Initializes web content feature.

  3. vision read section {section_name}
    Reads all the paragraphs from the section {section_name}.

  4. vision read para {para_no} of section {section_name}
    Only reads the paragraph with number {para_number} from the section with name {section_name}.

  5. vision read all images of section {section_name}
    Describes all the images from a particular section.

  6. vision read image {image_number} of section {section_name}
    Describes a specific image from section with name {section_name} and having number as {image_number}

  7. vision paragraph
    Tells what commands to use for reading out all the paragraphs or some particular paragraph from the specific section.

  8. vision image
    Tells what commands to use for describing all the images or a particular image from the specific section.
